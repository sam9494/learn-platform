# Week 3：抽樣與估計（每日詳細內容）

## Day 1：母體與樣本

### 學習目標
理解統計推論的基本框架

### 核心概念
- **母體 (Population)**：研究對象的全體
- **樣本 (Sample)**：母體中被抽出的一部分
- **參數 (Parameter)**：描述母體的量（如 μ、σ）— **未知但固定**
- **統計量 (Statistic)**：從樣本計算的量（如 x̄、s）— **已知但隨機**

### 為什麼需要抽樣？
- 母體太大無法全測（例：全台選民）
- 測量會破壞（例：燈泡壽命）
- 成本考量

### 抽樣方法
- **簡單隨機抽樣 (SRS)**：最基本
- **分層抽樣**：先分組再抽
- **系統抽樣**：每 k 個抽一個
- **群集抽樣**：抽「組」而非個體

### 偏差警告
選舉民調若只打市話 → 遺漏手機族群 → **選擇偏差**

### 練習
設計一個研究「台灣大學生睡眠時間」的抽樣計畫，討論潛在偏差

## Day 2：樣本統計量的分布

### 學習目標
理解「統計量本身也是隨機變數」

### 關鍵觀念
每次抽樣都會得到不同的 x̄ → **x̄ 是隨機變數**
→ 它有自己的分布（稱為**抽樣分布**）

### 樣本平均的性質
若 X₁, ..., Xₙ 獨立同分布 (iid)，且 E[Xᵢ] = μ、Var(Xᵢ) = σ²：

- **E[x̄] = μ**（不偏）
- **Var(x̄) = σ²/n**（樣本越大越穩定）
- **SD(x̄) = σ/√n**（標準誤差）

### 重要區別
- **標準差 (σ)**：單一觀測值的離散程度
- **標準誤差 (σ/√n)**：樣本平均的離散程度

### 直觀
- n 越大，x̄ 越接近 μ（**大數法則**）
- 但縮小速度是 √n，不是 n（所以樣本數常常要很大）

### 練習

## Day 3-4：中央極限定理（CLT）

### 學習目標
掌握統計學的「靈魂定理」

### 定理陳述
若 X₁, ..., Xₙ 獨立同分布，E[Xᵢ] = μ、Var(Xᵢ) = σ² < ∞：

當 n → ∞，
$$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0, 1)$$

### 白話解釋
**不管原始母體是什麼分布，只要 n 夠大，樣本平均的分布會趨近常態分布**

### 為什麼這麼重要？
- 我們很少知道母體分布
- 但 CLT 告訴我們樣本平均一定接近常態
- → 可以用常態分布做推論！

### 實務經驗法則
- n ≥ 30：一般情況 CLT 已可用
- 原始分布偏態嚴重時：需更大 n
- 原始分布對稱：n = 10 可能就夠

### CLT 的「魔法」展示
設想一個極端偏態分布（如指數分布）：
- 單一觀測值：嚴重偏態
- n = 5 的平均：仍有偏態
- n = 30 的平均：接近常態
- n = 100 的平均：幾乎完美常態

## Day 5：點估計

### 學習目標
理解如何「猜測」未知參數

### 估計量 (Estimator)
用樣本算出的值，用來估計母體參數
- 估計 μ → 用 x̄
- 估計 σ² → 用 s²

### 好的估計量應具備
1. **不偏性 (Unbiasedness)**
   - E[估計量] = 真實參數
   - 例：E[x̄] = μ ✓

2. **一致性 (Consistency)**
   - n → ∞ 時估計量收斂到真實參數
   - 大數法則保證樣本平均一致

3. **有效性 (Efficiency)**
   - 變異數越小越好
   - 同為不偏估計時，變異數小的比較好

### 為什麼樣本變異數要除以 (n-1)？
- 若除以 n：是有偏的（低估）
- 除以 (n-1)：修正後為不偏
- (n-1) 稱為**自由度**
- 直觀：用 x̄ 估計了一個參數，「用掉」一個自由度

### 常見估計量
| 參數 | 估計量 |
|------|--------|
| μ | x̄ |
| σ² | s² = Σ(xᵢ-x̄)²/(n-1) |
| p（比例）| p̂ = x/n |

## Day 6：區間估計與信賴區間

### 學習目標
從「點」估計升級到「範圍」估計

### 為什麼要區間估計？
- 點估計不會完全正確
- 區間估計給出「合理範圍」與「信心水準」

### 信賴區間的構造（以 μ 為例）
當 σ 已知：
$$\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$$

當 σ 未知（實務常見）：
$$\bar{x} \pm t_{\alpha/2, n-1} \cdot \frac{s}{\sqrt{n}}$$

### 95% 信賴區間的正確解讀
**錯誤解讀**：「真實 μ 有 95% 機率在這個區間內」
**正確解讀**：「若重複抽樣 100 次，建構出的 100 個區間中，約 95 個會包含真實 μ」

### 關鍵區別
- μ 是**固定未知值**，不是隨機變數
- 區間是**隨機的**（每次抽樣不同）
- 「95%」指的是方法的可靠性，不是單次結果

### 影響區間寬度的因素
- 信心水準 ↑ → 區間 ↑ （要更有把握當然要更寬）
- 樣本數 n ↑ → 區間 ↓
- 母體變異 σ ↑ → 區間 ↑

### t 分布 vs 常態分布
- 小樣本、σ 未知時用 t 分布
- t 分布比常態厚尾，反映「σ 未知」的額外不確定性
- n 大時 t 分布 → 常態分布

### 練習
n = 25, x̄ = 50, s = 10，建構 95% 信賴區間（t 值約 2.064）
