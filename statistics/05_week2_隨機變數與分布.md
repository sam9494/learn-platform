# Week 2：隨機變數與分布（每日詳細內容）

## Day 1：隨機變數的定義

### 學習目標
理解隨機變數不是「變數」，而是一個**函數**

### 核心概念
- **隨機變數 (Random Variable)**：從樣本空間映射到實數的函數
  - 符號：X: Ω → ℝ
- **為什麼需要？** 把抽象事件轉為數字，方便數學運算

### 例子
擲兩顆骰子：
- 樣本空間有 36 個結果
- 定義 X = 「點數和」
- X 把 {(1,1), (1,2)...} 映射到 {2, 3, ..., 12}

### 離散 vs 連續
- **離散**：取值可數（整數、有限集）
  - 例：擲骰子點數、一天來客數
- **連續**：取值不可數（實數區間）
  - 例：身高、等車時間

### 練習
定義一個自己的隨機變數，寫出樣本空間與映射規則

## Day 2：PMF 與 PDF

### 學習目標
區分離散與連續的機率描述方式

### 機率質量函數 (PMF)
- 離散隨機變數使用
- p(x) = P(X = x)
- 性質：
  - p(x) ≥ 0
  - Σ p(x) = 1

### 機率密度函數 (PDF)
- 連續隨機變數使用
- f(x) ≠ P(X = x)！
- P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx

### 關鍵觀念
**連續隨機變數在單點的機率是 0**
- P(X = 1.5) = 0
- 但 f(1.5) 可以是任意正數
- PDF 的值不是機率，是「密度」

### 累積分布函數 (CDF)
- F(x) = P(X ≤ x)
- 對離散、連續都適用
- 恆為非遞減、右連續

給一個簡單 PDF f(x) = 2x（0 ≤ x ≤ 1），求：
- P(X < 0.5)
- F(x) 的表達式

## Day 3：期望值與變異數

### 學習目標
掌握兩個最重要的統計量及其數學性質

### 期望值 E[X]
- 離散：E[X] = Σ x · p(x)
- 連續：E[X] = ∫ x · f(x) dx
- 直觀：長期平均值

### 期望值的線性性質（超重要）
**E[aX + bY] = aE[X] + bE[Y]**
- 即使 X、Y 不獨立也成立！
- 這是許多證明的核心工具

### 變異數 Var(X)
- 定義：Var(X) = E[(X − μ)²]
- 計算公式：Var(X) = E[X²] − (E[X])²
- 標準差：σ = √Var(X)

### 變異數的性質
- Var(aX + b) = a²Var(X)
- Var(X + Y) = Var(X) + Var(Y) **只在獨立時成立**

- E[X] = ?（答：3.5）
- Var(X) = ?（提示：先算 E[X²]）

## Day 4：重要離散分布

### 伯努利分布 Bernoulli(p)
- 一次試驗，成功機率 p
- E[X] = p, Var(X) = p(1-p)

### 二項分布 Binomial(n, p)
- n 次獨立伯努利試驗的成功次數
- PMF：P(X = k) = C(n,k) pᵏ(1-p)ⁿ⁻ᵏ
- E[X] = np, Var(X) = np(1-p)

### 泊松分布 Poisson(λ)
- 單位時間/空間內事件發生次數
- PMF：P(X = k) = (λᵏ e⁻ᵏ) / k!
- E[X] = Var(X) = λ
- **應用**：來客數、網站點擊、放射性衰變

### 幾何分布 Geometric(p)
- 第一次成功前的失敗次數
- E[X] = (1-p)/p

### 二項 → 泊松的近似
當 n 大、p 小、np 適中 → 二項近似泊松（λ = np）

## Day 5：重要連續分布（上）

### 均勻分布 Uniform(a, b)
- PDF：f(x) = 1/(b-a), a ≤ x ≤ b
- E[X] = (a+b)/2
- Var(X) = (b-a)²/12

### 指數分布 Exponential(λ)
- PDF：f(x) = λe⁻λˣ, x ≥ 0
- E[X] = 1/λ, Var(X) = 1/λ²
- **無記憶性**：P(X > s+t | X > s) = P(X > t)
- **應用**：等待時間、壽命

### 指數與泊松的關係
- 泊松過程中，事件之間的**間隔時間**服從指數分布
- 兩者是一體兩面

### 練習

- 平均壽命？
- 用超過 5 年的機率？

## Day 6：重要連續分布（下）— 常態分布

### 常態分布 N(μ, σ²)
- PDF：f(x) = (1/√(2πσ²)) exp(-(x-μ)²/2σ²)
- 鐘形曲線、對稱於 μ
- 完全由 μ、σ 決定

### 標準常態 Z ~ N(0, 1)
- 任何 X ~ N(μ, σ²) 可標準化：Z = (X - μ)/σ

### 68-95-99.7 法則
- P(|X - μ| < σ) ≈ 68%
- P(|X - μ| < 2σ) ≈ 95%
- P(|X - μ| < 3σ) ≈ 99.7%

### 為什麼常態分布無所不在？
- **中央極限定理**（Week 3 會詳講）
- 許多自然現象是多個獨立因素相加的結果

### 查表技巧
- Z 表查累積機率 Φ(z)
- P(a < Z < b) = Φ(b) - Φ(a)
- 對稱性：Φ(-z) = 1 - Φ(z)
