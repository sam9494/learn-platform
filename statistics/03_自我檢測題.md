# 月底自我檢測題

## 使用方式
- 每週末挑當週相關題目測試
- 月底全部做一次，檢視學習成果
- 不會做的題目 → 回去補對應章節

## Week 1：機率基礎

### 觀念題
1. 「互斥」與「獨立」有什麼不同？各舉一例。
2. 寫出機率三公理，並從中推導 P(Aᶜ) = 1 − P(A)。
3. 條件機率 P(A|B) 的幾何直觀是什麼？
4. 貝氏定理中的「先驗」、「概似」、「後驗」分別代表什麼？

### 計算題
5. 擲兩顆公平骰子，已知點數和為偶數，求點數和為 8 的條件機率。
6. 某城市下雨天機率 30%，下雨時塞車機率 80%，不下雨時塞車機率 20%。
   - (a) 今天塞車的機率？
   - (b) 若今天塞車，下雨的機率？
7. **蒙提霍爾問題**：三扇門後有一台車兩隻羊。你選了 A 門。主持人（知道答案）打開 B 門顯示羊。換到 C 門的獲勝機率是多少？

## Week 2：隨機變數與分布

### 觀念題
1. PMF 與 PDF 有什麼差別？為什麼連續隨機變數在單點的機率是 0？
2. 期望值的線性性質是什麼？即使 X、Y 不獨立也成立嗎？
3. 變異數 Var(X) 的三種等價定義各是什麼？
4. 常態分布為什麼「自然」？（提示：中央極限定理）

### 計算題
5. 設 X ~ Binomial(10, 0.3)，求 E[X]、Var(X)、P(X = 3)。
6. 設 X ~ N(0, 1)，用查表求：
   - (a) P(X < 1.96)
   - (b) P(−1 < X < 1)
   - (c) P(|X| > 2)
7. 泊松分布 λ = 3，求至少發生一次的機率。

## Week 3：抽樣與估計

### 觀念題
1. 用自己的話解釋**中央極限定理**。
2. 「不偏估計量」的定義？樣本變異數為何要除以 (n−1) 而非 n？
3. 95% 信賴區間的**正確**解讀是什麼？哪些解讀是錯的？
4. 樣本數變大，信賴區間寬度如何變化？為什麼？

### 計算題
5. 從母體隨機抽 100 個樣本，樣本平均 50、樣本標準差 10。求母體平均的 95% 信賴區間。
6. 若要讓信賴區間寬度縮小一半，樣本數要變幾倍？

## Week 4：假設檢定與迴歸

### 觀念題
1. 虛無假設 H₀ 與對立假設 H₁ 的設定原則？
2. p 值的**正確定義**是什麼？以下哪些解讀是錯的：
   - (a) H₀ 為真的機率
   - (b) 資料出現「極端程度」的機率（假設 H₀ 為真）
   - (c) 研究結果錯誤的機率
3. 第一型錯誤與第二型錯誤分別是什麼？哪個比較嚴重？（取決於情境）
4. 線性迴歸的最小平方法在最小化什麼？為什麼不用絕對值？

### 計算題
5. 某工廠聲稱產品重量平均 500g。抽樣 36 個，樣本平均 498g、標準差 6g。
   - 在 α = 0.05 下檢定工廠聲稱。
   - 算出 p 值並解讀。
6. 給定五個點 (1,2), (2,4), (3,5), (4,4), (5,5)，求簡單線性迴歸的 β₀、β₁。

## 整合應用題

### 題目 A：貝氏思維
某新冠快篩敏感度 85%、特異度 98%。若社區盛行率 2%，陽性結果真的有病的機率？
若盛行率升到 20%，答案如何變化？畫圖思考先驗機率的影響。

### 題目 B：中央極限定理驗證
假設有個非常偏態的分布（例如指數分布），若從中重複抽樣 n = 30，計算 1000 次樣本平均。
- 理論上這些樣本平均的分布長什麼樣？
- 若 n = 5 呢？
（這題未來可用程式驗證，但先用理論推理）

### 題目 C：檢定陷阱
某論文聲稱「實驗組比對照組表現好，p = 0.04」。請列出三個你會追問的問題。

## 自評標準

- **A 級**（全懂）：觀念題能用自己的話解釋，計算題 80% 以上會做
- **B 級**（大致通）：主要概念理解，計算題 60% 會做，知道哪裡需補強
- **C 級**（需加強）：計算可做但概念模糊，或反之
- **D 級**（重讀）：多數題目無法下手

不必追求一次 A 級，統計要回頭看好幾次才會真的懂。
