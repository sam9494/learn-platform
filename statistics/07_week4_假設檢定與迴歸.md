# Week 4：假設檢定與迴歸（每日詳細內容）

## Day 1：假設檢定的邏輯

### 學習目標
理解假設檢定的哲學，不只是套公式

### 基本框架
1. 設立**虛無假設 H₀**（通常是「無差異」「無效果」）
2. 設立**對立假設 H₁**（你想證明的）
3. 收集資料，計算檢定統計量
4. 判斷資料是否與 H₀ 矛盾
5. 決定：拒絕 H₀ 或不拒絕 H₀

### 關鍵哲學
- 我們**永遠不能「證明」H₀ 為真**
- 只能「拒絕 H₀」或「沒有足夠證據拒絕 H₀」
- 類似刑事審判：推定無罪，除非證據超越合理懷疑

### 法庭類比
- H₀：被告無罪
- H₁：被告有罪
- 拒絕 H₀（有罪判決）需要強力證據
- 「無罪釋放」不等於「真的沒做」

### 假設設定原則
- H₀ 通常帶 = 號
- 想要推翻的放 H₀，想要支持的放 H₁
- 舉證責任在對立假設

### 練習
為以下情境寫出 H₀ 和 H₁：
1. 新藥是否比舊藥有效？
2. 骰子是否公平？
3. 男女平均薪資是否不同？

## Day 2：p 值與顯著水準

### 學習目標
正確理解 p 值（這是最常被誤解的統計概念）

### p 值的正確定義
**假設 H₀ 為真，觀察到目前資料或更極端資料的機率**

### 常見誤解（請逐一釐清）
❌ 「p 值 = H₀ 為真的機率」
❌ 「p 值 = H₁ 為假的機率」
❌ 「1 - p 值 = 結論正確的機率」
❌ 「p 值越小，效應越大」

✓ p 值只是「若 H₀ 為真，這資料有多不尋常」

### 顯著水準 α
- 事先設定（通常 0.05）
- 規則：p < α → 拒絕 H₀

### p = 0.05 的意義
- 若 H₀ 為真，約 5% 的時候會看到這麼極端的資料
- 不等於「有 5% 機率搞錯」

### 單尾 vs 雙尾檢定
- 雙尾：H₁ 是 μ ≠ μ₀
- 單尾：H₁ 是 μ > μ₀ 或 μ < μ₀
- 雙尾更保守（同 α 下，單尾更容易拒絕 H₀）
- **事前決定**，不能看到資料才改

### 統計顯著 vs 實務顯著
- 樣本極大時，微小差異也可能 p < 0.05
- 但這差異可能完全沒有實務意義
- → 要同時看**效應量 (Effect Size)**

## Day 3：兩種錯誤與檢定力

### 學習目標
理解檢定的四種可能結果

### 混淆矩陣
|  | H₀ 為真 | H₀ 為假 |
|---|--------|--------|
| 不拒絕 H₀ | 正確 (1-α) | **第二型錯誤 β** |
| 拒絕 H₀ | **第一型錯誤 α** | 正確 (1-β，檢定力) |

### 第一型錯誤 (Type I)
- H₀ 為真但被拒絕
- 機率 = α（我們設定的顯著水準）
- 類比：無辜的人被定罪

### 第二型錯誤 (Type II)
- H₀ 為假但沒拒絕
- 機率 = β
- 類比：真兇被釋放

### 檢定力 (Power = 1 - β)
- H₀ 為假時成功拒絕的機率
- 影響因素：
  - 效應量越大 → 檢定力越高
  - 樣本數越大 → 檢定力越高
  - α 越大 → 檢定力越高（但代價是更多第一型錯誤）

### 哪個錯誤更嚴重？
**依情境而定**：
- 新藥測試：第一型更嚴重（無效藥上市）
- 癌症篩檢：第二型更嚴重（漏診致命）

### 樣本數規劃
實驗前要先算：為了達到想要的檢定力，需要多大樣本？
（這是實驗設計的重要步驟）

## Day 4：常見檢定（上）— t 檢定

### 單樣本 t 檢定
- 檢定母體平均 μ 是否等於某值 μ₀
- 檢定統計量：t = (x̄ - μ₀) / (s/√n)
- 自由度 = n - 1

### 兩樣本 t 檢定
檢定兩組平均是否相等：
- **獨立樣本**：兩組不同的人
- **配對樣本**：同一批人前後測量

### 例子：藥效檢定
H₀：μ_新藥 = μ_舊藥
H₁：μ_新藥 > μ_舊藥
抽 30 位患者，比較療效指標

### t 檢定的假設
1. 樣本獨立
2. 母體大致常態（或 n 夠大，CLT 保護）
3. 兩組變異數相等（可用 Welch's t-test 放寬）

## Day 5：常見檢定（下）— 卡方與 ANOVA

### 卡方適合度檢定
檢定觀察分布是否符合理論分布
- 例：擲骰子 600 次，每面各出現幾次？是否公平？

### 卡方獨立性檢定
檢定兩個類別變數是否獨立
- 例：性別 vs 投票傾向是否獨立？
- 用列聯表 (Contingency Table)

### 卡方統計量
$$\chi^2 = \sum \frac{(O - E)^2}{E}$$
- O：觀察次數
- E：期望次數（假設 H₀ 為真）

### ANOVA（變異數分析）
- 比較**三組以上**平均
- 為什麼不兩兩 t 檢定？→ 多重比較膨脹第一型錯誤
- 核心：組間變異 vs 組內變異的比值 (F 統計量)

### 無母數檢定（了解即可）
- 當資料不符合常態假設
- Mann-Whitney U、Wilcoxon、Kruskal-Wallis

## Day 6：簡單線性迴歸

### 學習目標
理解迴歸的核心：找出最佳直線

### 模型
$$Y = \beta_0 + \beta_1 X + \varepsilon$$
- β₀：截距
- β₁：斜率（X 每增 1，Y 平均增 β₁）
- ε：誤差項（假設 ~ N(0, σ²)）

### 最小平方法 (OLS)
最小化殘差平方和：
$$\text{SSR} = \sum (y_i - \hat{y}_i)^2$$

### 為什麼用平方而非絕對值？
- 數學上可微、有閉式解
- 對大誤差懲罰更重
- 統計性質較好（BLUE：最佳線性不偏估計）

### 公式（單變量）
$$\hat{\beta}_1 = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2}$$
$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$

### 判定係數 R²
- 定義：R² = 1 - SSR/SST
- 解讀：X 解釋了 Y 多少百分比的變異
- 範圍 [0, 1]
- R² 高不等於「因果關係」

### 迴歸的基本假設
1. 線性關係
2. 殘差獨立
3. 殘差變異數齊一（同方差性）
4. 殘差常態

### 常見陷阱
- **外推危險**：訓練範圍外的預測不可信
- **相關 ≠ 因果**
- **離群值**對迴歸線影響大
- **安斯庫姆四重奏**：不同資料可有相同 R²（提醒：永遠畫圖看）
