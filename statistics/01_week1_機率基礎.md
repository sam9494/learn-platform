# Week 1：機率基礎（每日詳細內容）

## Day 1：樣本空間與事件

### 學習目標
理解機率問題的「舞台」怎麼設定

### 核心概念
- **樣本空間 (Ω)**：所有可能結果的集合
  - 擲一顆骰子：Ω = {1, 2, 3, 4, 5, 6}
- **事件 (Event)**：樣本空間的子集
  - 「擲出偶數」= {2, 4, 6}
- **事件運算**：聯集 (∪)、交集 (∩)、補集 (Aᶜ)

### 練習
1. 擲兩顆骰子，寫出樣本空間（提示：36 個元素）
2. 定義事件 A = 點數和為 7，列出 A 的所有元素

## Day 2：機率公理

### 學習目標
理解機率的數學定義（Kolmogorov 公理）

### 三大公理
1. **非負性**：P(A) ≥ 0
2. **規範性**：P(Ω) = 1
3. **可加性**：若 A、B 互斥，P(A ∪ B) = P(A) + P(B)

### 推論
- P(Aᶜ) = 1 − P(A)
- P(A ∪ B) = P(A) + P(B) − P(A ∩ B)（容斥原理）

### 練習
證明：若 A ⊆ B，則 P(A) ≤ P(B)

---

## Day 3：條件機率

### 學習目標
理解「在已知某事件發生下」的機率

### 定義
$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

### 直觀理解
條件機率 = 把樣本空間「縮小」到 B 裡面

### 經典例子
家裡有兩個小孩，已知至少一個是女孩，兩個都是女孩的機率？
（答案 1/3，不是 1/2，想想為什麼）

## Day 4：獨立性

### 學習目標
區分「獨立」與「互斥」（最常混淆）

### 定義
A、B 獨立 ⟺ P(A ∩ B) = P(A) × P(B)
等價於：P(A|B) = P(A)

### 關鍵區別
| | 獨立 | 互斥 |
|---|------|------|
| 定義 | 一方發生不影響另一方 | 不可能同時發生 |
| 交集 | P(A∩B) = P(A)P(B) | P(A∩B) = 0 |
| 例子 | 連擲兩次硬幣 | 擲一次骰子，擲出 1 vs 擲出 2 |

### 重要觀念
互斥 + 兩者機率都 > 0 ⟹ **不獨立**

## Day 5：貝氏定理（上）

### 學習目標
推導貝氏定理並理解各項意義

### 公式
$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

### 術語
- P(A)：**先驗機率** (prior)
- P(B|A)：**概似度** (likelihood)
- P(A|B)：**後驗機率** (posterior)
- P(B)：**證據** (evidence，正規化常數)

### 推導
從條件機率定義出發：
- P(A|B)P(B) = P(A ∩ B)
- P(B|A)P(A) = P(A ∩ B)
- 兩式相等 → 貝氏定理

## Day 6：貝氏定理（下）— 實戰應用

### 經典問題：醫療檢測
某疾病盛行率 1%，檢測敏感度 99%，特異度 95%。
若檢測陽性，真的有病的機率？

**解**：
- P(病) = 0.01
- P(陽|病) = 0.99
- P(陽|無病) = 0.05
- P(陽) = 0.99×0.01 + 0.05×0.99 = 0.0594
- P(病|陽) = (0.99 × 0.01) / 0.0594 ≈ **16.7%**

### 驚人結論
即使檢測準確度 99%，陽性結果仍只有 17% 真的有病！

### 為什麼？
先驗機率很低 → 偽陽性淹沒真陽性
