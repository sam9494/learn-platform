## 已掌握的核心觀念

### 監督式學習基礎
- 訓練 vs 預測 兩階段差別（訓練需要答案 y、預測只給 x）
- 特徵 x、權重 w、偏移 b、預測 ŷ / p 的角色
- 多特徵：`ŷ = w₁·x₁ + w₂·x₂ + ... + b`

### Sigmoid / 邏輯回歸
- 公式：`σ(x) = 1 / (1 + e^(−x))`，把任意實數壓到 0~1
- S 形曲線、飽和區梯度消失、現代隱藏層改用 ReLU
- 決策邊界落在 `x = −b/w`
- 二元分類輸出層仍是標準選擇

### 線性回歸
- 公式：`ŷ = w·x + b`（邏輯回歸拿掉 sigmoid）
- 每個 w 的可解釋性高：「該特徵每加 1 單位，預測值變動多少」
- 有「封閉解 / Normal Equation」可以一步解出（邏輯回歸沒有）

### Loss Function
- 角色：把「預測 vs 真實答案」變成一個可比較、可微分的數字
- **MSE**（回歸）：`(1/N)·Σ(ŷ−y)²`，平方放大大誤差
- **Binary Cross-Entropy**（二元分類）：`−[y·log(p) + (1−y)·log(1−p)]`
  - 直覺：當 y=1 → `−log(p)`；當 y=0 → `−log(1−p)`
  - 越自信的錯誤被懲罰越重
- 為什麼不用正確率當 loss：不可微、無法分辨「差一點」和「差超多」

### 梯度下降
- 大圖：猜 w,b → 算預測 → 算 loss → 算梯度 → 更新 w,b → 重複
- 邏輯回歸梯度：`∂L/∂w = (p−y)·x`、`∂L/∂b = (p−y)`
- 更新規則：`w ← w − η·梯度`，η 是學習率（0.01~0.1）

### ML 問題類型全景
- 監督式（分類 / 回歸）、非監督（分群 / 降維 / 異常 / 關聯）、自監督、強化學習
- 判斷自己的問題：有答案資料？→ 監督；答案是類別還是數字？→ 分類 vs 回歸
