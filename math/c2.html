積分入門

Integration — 用「面積」累積總量

1這一課在解決什麼問題？

核心問題：「給定一個變化率，總共累積了多少？」
微分問「變化多快」，積分問「累積多少」。
在圖形上，積分就是曲線下的面積。

2從矩形逼近面積

怎麼算彎曲曲線下的面積？古希臘的辦法：用很多窄矩形堆起來逼近。

當矩形數量 n → ∞，矩形寬度 → 0，總和就是定積分，記作：

∫ab f(x) dx
用 10 個矩形逼近 ∫₀² x² dx
≈ 2.85
真實答案：8/3 ≈ 2.667

3微積分基本定理

核心結論：「微分和積分互為反操作。」
若 F'(x) = f(x)，則 ∫ab f(x) dx = F(b) − F(a)。

這是微積分最重要的定理（牛頓和萊布尼茲分別發現）。它讓「算面積」這件事變成「找反導數」，超級簡單。

例：算 ∫₀² x² dx

  1. 找 x² 的反導數：F(x) = x³/3（因為 (x³/3)' = x²）
  2. 代入上下限：F(2) − F(0) = 8/3 − 0 = 8/3
  3. 結果 ≈ 2.667 ✓

4常用積分公式

函數反導數（不定積分）
xⁿ （n ≠ -1）x^(n+1) / (n+1) + C
1/xln|x| + C
eˣeˣ + C
sin(x)−cos(x) + C
cos(x)sin(x) + C

C 是任意常數。因為常數的導數為 0，所以反導數差一個 C 也都對 — 這叫「不定積分」。

5真實應用

🚗

從速度算位移

對速度 v(t) 積分 = 走過的距離。速度表盤的累積就是行駛里程。

🌊

水流總量

水龍頭每秒出 v(t) 升水 → 積分 = 水缸總水量。

⚡

用電總量

家裡某時刻耗電 P(t) → 積分 = 一個月的總用電（kWh）。

📊

機率密度

對機率密度函數積分 = 落在某區間的機率。

🏗️

不規則面積

湖泊面積、土地面積，用積分精確算。

🎢

物體體積

旋轉體體積、不規則物體都靠積分。

🎓 重點

  • 定積分 = 曲線下面積。
  • 用反導數算積分，是微積分基本定理的功勞。
  • 次方規則反過來：∫xⁿdx = x^(n+1)/(n+1) + C。
  • 微分問「變化率」，積分問「累積量」 — 兩者是同一硬幣的兩面。
