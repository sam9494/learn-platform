微分初探

Differentiation — 用「斜率」描述變化

1這一課在解決什麼問題？

核心問題：「在某個瞬間，量變化得多快？」
平均速度好算（總距離 ÷ 總時間），但「此時此刻」的速度呢？ 微分就是用來算這個「瞬間變化率」的工具。它在圖形上表現為切線斜率。

2從割線到切線

取曲線上兩點 A、B，連線是割線，斜率 = (y₂−y₁) / (x₂−x₁)。

當 B 越來越靠近 A，割線會「逼近」一條剛好碰一點的線 — 那就是切線。

切線的斜率就是該點的瞬間變化率，記作 f'(x) 或 dy/dx。

在 x = 1.0 處
f'(x) = 2.000
（也就是切線的斜率）

四個運算規則

  • 常數倍：(c · f)' = c · f'
  • 加減：(f ± g)' = f' ± g'
  • 乘積：(f · g)' = f'g + fg'
  • 連鎖律：f(g(x))' = f'(g(x)) · g'(x)（最常用、最容易錯）

4微分能做什麼？

⚡

找瞬間速度

位置函數的導數 = 速度。速度的導數 = 加速度。

📈

找最大 / 最小值

極值點的斜率 = 0。把 f'(x) = 0 解出來就找到候選點。

📐

畫切線

已知斜率與一點 → 點斜式 y - y₀ = m(x - x₀) 寫出切線。

🤖

機器學習

梯度下降法（Gradient Descent）就是用導數找誤差最低點。

5真實例子

🚗

汽車速度表

顯示的是「此刻」的速度，不是平均。這是位置對時間的導數。

📊

股價分析

「現在漲得多快」就是價格對時間的導數，技術分析的基礎。

🏭

邊際成本

多生產 1 單位多花多少錢？= 總成本函數的導數。

🎓 重點

  • 微分的本質是「瞬間變化率」，幾何上 = 切線斜率。
  • 次方規則 (xⁿ)' = n·xⁿ⁻¹ 是入門必背。
  • 找極值就解 f'(x) = 0。
  • 下一課的積分是它的「反操作」。
