矩陣與線性變換

Matrices — 把空間「拉、轉、壓」的指令

1這一課在解決什麼問題？

核心問題：「怎麼用一個物件，描述空間整體的變形？」
矩陣不是「一堆數字」，它是「變形指令」 — 把整個空間旋轉、縮放、剪切，全都可以濃縮成一個 2×2 矩陣。

2矩陣的基本長相

一個 2×2 矩陣寫成：

ab cd

關鍵想法：矩陣決定基底向量去哪

  • 第 1 欄 (a, c) = î = (1, 0) 變換後的位置
  • 第 2 欄 (b, d) = ĵ = (0, 1) 變換後的位置

知道了基底走向哪，整個空間怎麼變就確定了。

3互動：拖動矩陣，看空間變形

虛線網格是原始空間，實線網格是變形後。藍 / 粉箭頭是 î、ĵ 變換後的位置。

det = 1.00 ← 行列式 = 變形後面積的倍率

4矩陣 × 向量

矩陣作用在向量上：

[a b; c d] · (x, y) = (ax + by, cx + dy)

常見變換

變換矩陣效果
恆等[1 0; 0 1]什麼都不變
放大 k 倍[k 0; 0 k]整個空間放大
旋轉 θ[cosθ -sinθ; sinθ cosθ]逆時針旋轉
水平剪切[1 k; 0 1]把方塊變平行四邊形
x 軸反射[1 0; 0 -1]上下翻

5真實應用

🎮

3D 遊戲

所有物體的旋轉、縮放、移動都用矩陣計算。GPU 為這量身打造。

📷

相機透視

3D 場景投影到 2D 螢幕的計算。

🤖

神經網路

每一層 = 一次矩陣乘法 + 非線性函數。

🔍

Google PageRank

網頁排名算法用的是巨大矩陣的特徵向量。

📈

主成分分析

從高維資料中找出主要方向，用矩陣分解完成。

📐

工程結構

橋樑、建築的應力分析離不開矩陣計算。

🎓 重點

  • 矩陣 = 線性變換的具體表示。
  • 矩陣的欄就是基底向量變換後的位置。
  • 行列式 det = ad - bc，告訴你變換後面積放大幾倍（負號表示翻面）。
  • 連續變換 = 矩陣相乘（注意順序，AB ≠ BA）。
