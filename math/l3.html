Algebra · L3

解方程式

Solving Equations — 把未知數從等式中「揪」出來

1這一課在解決什麼問題？

核心問題：「等號兩邊已知一個關係，怎麼算出未知的數？」
生活中處處是「未知」：手機分期還剩多少？打折後是多少？方程式就是把未知用 x 代替， 然後從等式中還原它的值。

🔑 黃金原則

等式像天平，左右一定要平衡。
兩邊同時做一樣的操作（同加、同減、同乘、同除），等式仍成立。

2一元一次方程式

形如 ax + b = c。目標：把 x 單獨留在一邊。

例：解 2x + 5 = 13

2x + 5 = 13
原式
2x + 5 − 5 = 13 − 5
兩邊同減 5（消掉常數）
2x = 8
化簡
2x ÷ 2 = 8 ÷ 2
兩邊同除 2（消掉係數）
x = 4 ✓
代回驗算：2(4) + 5 = 13 ✓

視覺化：天平

3一元二次方程式

形如 ax² + bx + c = 0。有 3 種解法，挑最快的用：

①

因式分解

湊出 (x − p)(x − q) = 0，則 x = p 或 x = q。
例：x² − 5x + 6 = 0 → (x−2)(x−3) = 0 → x = 2 或 3

②

公式解

x = (−b ± √(b² − 4ac)) / 2a

萬用，但要小心代符號。

③

配方法

湊成 (x + p)² = q，再開平方。也能找到頂點。

判別式 Δ = b² − 4ac

  • Δ > 0：兩個不同的實數解
  • Δ = 0：一個重根（拋物線剛好碰 x 軸）
  • Δ < 0：沒有實數解（拋物線不碰 x 軸）

4真實世界例子

💰

分攤費用

5 個人吃 1500 元，每人 x 元，列式 5x = 1500，x = 300。

🏃

追上的時間

哥哥每分走 80m，弟弟走 60m，弟弟先走 5 分鐘。何時追上？列等式解 x。

🎯

拋物線落點

球的軌跡 y = -5x² + 20x，落地時 y = 0 → 解二次方程。

🎓 重點

  • 解方程式 = 「移項 + 化簡」，本質是天平兩邊同步操作。
  • 移項時記得變號（左邊加 5 移到右邊變減 5）。
  • 二次方程優先試因式分解，分解不出來再用公式。
  • 解完一定要代回驗算。
