Algebra · L4

指數與對數

Exponents & Logarithms — 一個描述「倍數成長」，一個把「倍數」變成「次方」

1從一個真實問題開始

2020 年 3 月，某城市每日確診數：1, 2, 4, 8, 16, 32 ……
你怎麼知道下一天會是 64？再下一天 128？
你需要的工具叫 指數函數。
但如果問題反過來——「現在 1024 人，是第幾天？」
你需要的就是 對數。

本課要回答四件事：

  • 指數 aⁿ 為什麼有那五條運算律？（不是死背，是推出來的）
  • 對數是什麼？它和指數什麼關係？
  • 對數律 log(xy) = log x + log y 為什麼成立？
  • 現實怎麼用：複利、疫情、地震、藥物、演算法。

2指數：「重複乘」的速記

從定義開始：

aⁿ = a × a × a × … × a　（n 個 a 相乘）

這裡 a 叫底數（base），n 叫指數（exponent）。

2.1　五條指數律的推論

這些不是規定，而是從定義展開後自然得到的結果。

▼ 規則 1：aᵐ × aⁿ = aᵐ⁺ⁿ
aᵐ × aⁿ = (a·a·…·a) × (a·a·…·a)
　　　　　　 m 個　　　　　 n 個
　　　 = a·a·a·…·a　（共 m + n 個）
　　　 = aᵐ⁺ⁿ ✓
數 a 的個數加起來，就是新的指數。
▼ 規則 2：aᵐ ÷ aⁿ = aᵐ⁻ⁿ
aᵐ / aⁿ = (a·a·…·a) / (a·a·…·a)
　　　　　　 m 個　　　　　 n 個
分子分母可以「約掉」n 對 a：
　　　 = a·a·…·a　（剩 m − n 個）= aᵐ⁻ⁿ ✓
▼ 規則 3：(aᵐ)ⁿ = aᵐⁿ
(aᵐ)ⁿ = aᵐ × aᵐ × … × aᵐ　（n 個 aᵐ）
　　　 用規則 1：指數加總，每個是 m，加 n 次
　　　 = aᵐ⁺ᵐ⁺…⁺ᵐ = aᵐⁿ ✓
▼ 規則 4：a⁰ = 1（為什麼？）
由規則 2：aⁿ / aⁿ = aⁿ⁻ⁿ = a⁰
但同時 aⁿ / aⁿ = 1（任何數除以自己 = 1）
兩式相等 → a⁰ = 1 ✓
這不是規定，是為了讓規則 2 在 m = n 時不矛盾。
▼ 規則 5：a⁻ⁿ = 1/aⁿ（負次方的意義）
由規則 1：aⁿ × a⁻ⁿ = aⁿ⁺⁽⁻ⁿ⁾ = a⁰ = 1
→ a⁻ⁿ = 1/aⁿ ✓
負指數 = 倒數。這也不是規定，是為了讓規則 1 對負數也成立。

3對數：反過來問「乘幾次？」

定義　log_a b = c　⟺　aᶜ = b
白話：「a 自己乘幾次會等於 b？答案就是 c。」

3.1　從表格直接看

指數式對數式讀法
2³ = 8log₂ 8 = 32 乘 3 次得 8
10² = 100log₁₀ 100 = 210 乘 2 次得 100
5⁰ = 1log₅ 1 = 05 乘 0 次（什麼都沒乘）= 1
2⁻¹ = 0.5log₂ 0.5 = −12 乘 −1 次（取倒數）= 0.5
2¹⁰ = 1024log₂ 1024 = 101024 是 2 翻倍 10 次來的

3.2　三種常用底數

  • log₁₀（常用對數）：常省略寫成 log。地震、聲音、pH 都用它。
  • logₑ（自然對數）：寫成 ln，e ≈ 2.71828。微積分、連續成長、機率都用。
  • log₂：用於電腦科學（資訊理論、複雜度分析）。

3.3　四條對數律的推論

所有對數律都是從指數律翻譯過來的。看推論你就再也不用背。

▼ 規則 1：log(xy) = log x + log y　（乘變加）
設 log x = m, log y = n（同底，省略下標）
由定義：x = aᵐ, y = aⁿ
兩式相乘：xy = aᵐ × aⁿ = aᵐ⁺ⁿ　（用了指數律 1）
把這一步翻譯回對數：log(xy) = m + n = log x + log y ✓
這就是「對數把乘法變加法」的根源——古代沒有計算機時，工程師用對數表把難算的乘法變成查表加法。
▼ 規則 2：log(x/y) = log x − log y　（除變減）
同樣設 x = aᵐ, y = aⁿ
x/y = aᵐ⁻ⁿ　（指數律 2）
→ log(x/y) = m − n = log x − log y ✓
▼ 規則 3：log(xⁿ) = n · log x　（次方變乘）
設 log x = m，所以 x = aᵐ
xⁿ = (aᵐ)ⁿ = aᵐⁿ　（指數律 3）
→ log(xⁿ) = mn = n · log x ✓
這條最有用——它讓我們從指數中「拉下」變數。下面解疫情題會用到。
▼ 規則 4：換底公式 log_a b = log_c b / log_c a
設 log_a b = x，意思是 aˣ = b
兩邊取 log_c（任何底數都可以）：log_c(aˣ) = log_c b
用規則 3 把 x 拉下來：x · log_c a = log_c b
→ x = log_c b / log_c a ✓
這就是為什麼計算機只要 ln 和 log 兩個鍵，就能算任何底的對數。

4互動 1：指數與對數互為反函數

兩條曲線以 y = x 為軸對稱——這就是「反函數」的視覺意義。

y = bˣ（指數）
y = log_b(x)（對數）
--- y = x（對稱軸）

5互動 2：為什麼疫情圖要用 log 刻度？

同一份「每日確診人數」資料（每天 ×1.5 倍），切換 y 軸的刻度看看：

實際確診人數
→ 線性：曲線爆炸式成長
▼ 為什麼指數曲線在 log 軸會變直線？
假設疫情是 y = a · bˣ（每天 ×b 倍）
兩邊取 log：log y = log(a · bˣ) = log a + log(bˣ)
用對數律 3：log y = log a + x · log b
令 Y = log y，這就變成 Y = log a + (log b) · x
這是一條直線！斜率 = log b，截距 = log a。
所以現實做法：把資料畫在 log 軸，做線性迴歸，得到斜率 m → 反推 b = 10ᵐ，就找出每天倍率。

6真實世界例子（含計算）

💰 例 1　複利：e 為什麼出現？
本金 1 元，年利率 100%，一年後是多少？
每年算一次：1 × (1 + 1)¹ = 2
每半年算一次（半年利率 50%）：1 × (1 + 0.5)² = 2.25
每月算一次：(1 + 1/12)¹² ≈ 2.613
每天算：(1 + 1/365)³⁶⁵ ≈ 2.7146
每秒、每瞬間算（連續複利）：極限值 ≈ 2.71828… = e
這就是 e 的由來——它是「複利切到無限細」的成長極限。所以連續複利公式是 P · eʳᵗ。
🦠 例 2　疫情：從每日 ×1.3 倍，到「幾天翻一倍」？
每天確診人數變 1.3 倍。需要幾天才會翻倍（變 2 倍）？
要找 t 使得 1.3ᵗ = 2
兩邊取 log：t · log 1.3 = log 2　（對數律 3）
t = log 2 / log 1.3 ≈ 0.301 / 0.114 ≈ 2.64 天
結論：每天 +30% 的疫情，每 2.6 天就翻倍。 一週多一點 → 8 倍；兩週 → 64 倍。這就是為何防疫要快。
🌋 例 3　地震：規模 7 比 5 強多少倍？
芮氏規模定義：M = log₁₀(A/A₀)，A 是地震波振幅。
規模 7：A₇/A₀ = 10⁷
規模 5：A₅/A₀ = 10⁵
相比：A₇ / A₅ = 10⁷ / 10⁵ = 10² = 100 倍
規模差 2 → 振幅差 100 倍；釋放能量約差 1000 倍（因能量 ∝ A^1.5）。
🧪 例 4　pH：為什麼可樂比水酸 10000 倍？
pH 定義：pH = −log₁₀[H⁺]，[H⁺] 是氫離子濃度。
純水 pH = 7　→　[H⁺] = 10⁻⁷ mol/L
可樂 pH = 3　→　[H⁺] = 10⁻³ mol/L
比值：10⁻³ / 10⁻⁷ = 10⁴ = 10000 倍
pH 看起來只差 4，氫離子濃度卻差一萬倍。這就是對數刻度——讓人類能用一個個位數，描述跨越 14 個數量級的酸鹼度。
💊 例 5　藥物：吃下後 12 小時還剩多少？
某藥半衰期 4 小時（每 4 小時減半）。吃下 200 mg，12 小時後剩多少？
12 小時 = 3 個半衰期
200 × (1/2)³ = 200 × 1/8 = 25 mg
要降到 1 mg 以下需要幾小時？(1/2)ⁿ × 200 < 1 → n > log₂ 200 ≈ 7.6
需要 8 個半衰期 = 32 小時。這就是為何醫生說「停藥後一兩天才完全代謝」。
💻 例 6　二分搜尋：100 萬筆資料只要 20 步
在排序好的陣列中找一個數，每次砍一半。資料量 N 時最多比幾次？
每次切半，剩餘 N → N/2 → N/4 → … → 1。
砍 k 次後剩 1：N / 2ᵏ = 1 → k = log₂ N
N = 1,000,000 時：log₂ 1000000 ≈ 19.9 → 最多 20 步
N 翻一倍只多 1 步。Google 搜尋上百億筆資料只比 ~33 步，就靠這個。

7動手檢測

Q1.　log₂ 32 = ?
2⁵ = 32，所以 log₂ 32 = 5。
Q2.　如果 log₁₀ 2 ≈ 0.301，那 log₁₀ 200 = ?
log(200) = log(100 × 2) = log 100 + log 2 = 2 + 0.301 = 2.301（對數律 1）。
Q3.　每天感染數變 2 倍。從 1 人到超過 100 萬人需要幾天？
2ⁿ ≥ 1,000,000 → n ≥ log₂ 10⁶ ≈ 19.9 → 第 20 天就破百萬。指數成長的可怕之處。
Q4.　某人聽 60 dB 的對話，旁邊飛機起飛 120 dB。飛機聲是對話的多少倍？
分貝 +10 = 強度 ×10。差 60 dB → 強度差 10⁶ = 100 萬倍。

🎓 一頁總結

  • 指數 aⁿ：a 自己乘 n 次。所有指數律都從這個展開推出。
  • 對數 log_a b：問「a 要乘自己幾次等於 b」。它和指數互為反函數。
  • 三大對數律：log(xy)=log x+log y、log(x/y)=log x−log y、log(xⁿ)=n·log x。
    本質上是把指數律翻譯回來。
  • 指數成長：「越來越快」（疫情、利息、人口）。
  • 對數成長：「越來越慢」（搜尋複雜度、資訊熵、感官刺激）。
  • log 刻度：把「乘」變成「加」，把「指數曲線」變成「直線」。
    這就是疫情圖、地震、pH 用對數的原因。
  • 實務心法：看到「翻倍幾次」「成長率」「半衰期」「等比」「強度差幾倍」就想到對數。
