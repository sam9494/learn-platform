機率基礎

Probability — 量化「不確定」

1這一課在解決什麼問題？

核心問題：「事情『有多可能』發生？」
生活充滿不確定 — 明天會下雨嗎？這場比賽誰贏？ 機率把「不確定」變成 0 ~ 1 之間的數字，讓我們可以理性比較與決策。

2機率的定義

古典機率：P(A) = 有利結果數 / 總結果數
前提是每個結果機會均等。

例子

  • 擲一顆公正骰子，得到 6 的機率：1/6 ≈ 0.167
  • 52 張牌抽到紅心：13/52 = 1/4
  • 擲兩個骰子和為 7：6/36 = 1/6

機率的範圍

  • P = 0 → 不可能發生
  • P = 1 → 必定發生
  • 0 < P < 1 → 可能發生，越大越可能

3互動：擲硬幣模擬

4三大重要規則

① 加法律（OR）

互斥事件：P(A ∪ B) = P(A) + P(B)
一般情況：P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

② 乘法律（AND）

獨立事件：P(A ∩ B) = P(A) × P(B)
例：擲兩次硬幣都正面 = 0.5 × 0.5 = 0.25

③ 條件機率

P(A | B) = P(A ∩ B) / P(B)

讀作「在 B 發生的條件下，A 發生的機率」。
例：抽到紅牌的條件下，是紅心的機率 = (13/52) / (26/52) = 1/2。

5真實應用

🎰

賭場賠率

賭場永遠贏的祕密：每個遊戲的「期望值」都是負的。

🌧️

天氣預報

「降雨機率 70%」是用大量歷史資料統計出來的。

🏥

醫學檢測

陽性結果不代表 100% 生病 — 條件機率（貝氏定理）告訴你真實機率。

🛡️

保險定價

保費是根據「事件機率 × 損失金額」算的期望理賠。

🤖

機器學習

分類器輸出的是「屬於某類的機率」。

📧

垃圾郵件過濾

樸素貝氏分類器：根據詞彙計算「是垃圾的條件機率」。

🎓 重點

  • 機率介於 0 和 1，是「不確定」的數值表示。
  • 獨立事件機率相乘，互斥事件機率相加。
  • 大數法則：實驗次數越多，實際比例越接近理論機率。
  • 直覺常騙人：學機率才能避免被誤導（如賭徒謬誤）。
