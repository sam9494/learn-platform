向量是什麼

Vectors — 有方向的量

1這一課在解決什麼問題？

核心問題：「光說『5』不夠 — 那 5 公里是『往哪』5 公里？」
日常生活很多量需要大小 + 方向：力、速度、位移、風。 向量就是用來表達這種「有方向的量」。

3互動：向量加法

拖兩個滑桿改變兩個向量，看「平行四邊形法則」如何給出和向量。

v₁ = (3, 2) + v₂ = (-1, 3) = v₁+v₂ = (2, 5)

4向量的基本運算

① 加法（首尾相接）

(a, b) + (c, d) = (a+c, b+d) — 各分量分別相加。

② 純量乘法（變長 / 反向）

k · (a, b) = (ka, kb)

  • k = 2 → 向量變兩倍長
  • k = 0.5 → 變一半
  • k = -1 → 反方向

③ 內積（Dot Product）

(a, b) · (c, d) = ac + bd

幾何意義：v · w = |v| |w| cos θ，θ 是兩向量夾角。

  • 內積 > 0 → 夾角 < 90°（同方向）
  • 內積 = 0 → 垂直（正交）
  • 內積 < 0 → 夾角 > 90°（反方向）

④ 長度（範數）

|v| = √(a² + b²) — 畢氏定理！

5真實應用

🚀

物理：力的合成

多個力同時作用，總效果 = 各力向量相加。

🛩️

飛行：風的影響

飛機速度 + 風速 = 實際對地飛行的向量。

🎮

遊戲：角色移動

按下方向鍵 → 給角色一個移動向量。

🤖

機器學習：特徵向量

每個資料點都是一個向量，相似度用內積算。

🧭

導航

GPS 路徑就是位置向量的時間序列。

🎨

電腦繪圖

每個像素的顏色 = (R, G, B) 三維向量。

🎓 重點

  • 向量 = 大小 + 方向。
  • 加法用「平行四邊形法則」或「首尾相接」。
  • 內積為 0 表示垂直，是線性代數的常用判斷。
  • 長度用畢氏定理算 √(x² + y²)。
